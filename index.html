Hasher API

The Hasher API allows for easy hashing of input values into hashvalues using a preferred hashing algorithm.

Available hashing algorithms

The used hashing algorithm heavily influences the generation of the hashvalue. By sending a GET request to ../hash/algorithms, a list of all the available hashing algorithms is returned.

Hashing a input value

The main function of this API is to generate hashvalues for given input values. This should be done corresponding to the following url:

../hash/algorithm/input

The above url will give an example response for the sha256 algorithm and an input value of "input". The output provided gives the chosen algorithm, the input value and the output value. Once again the required HTTP request is a GET request.
